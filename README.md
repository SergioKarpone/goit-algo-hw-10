# Висновки до Завдання 2: Обчислення інтеграла методом Монте-Карло

## Постановка задачі
Необхідно було обчислити значення визначеного інтеграла функції f(x) = x^2 на інтервалі [0, 2].

## Результати розрахунків

При виконанні коду було отримано два значення:
1.  **Точне значення** (за допомогою функції scipy.integrate.quad).
2.  **Наближене значення** (за допомогою власного алгоритму методу Монте-Карло з кількістю точок $N$).

Результати:

**Аналітичний метод (quad)** = 2.666666666666667 (еталонне значення)

**Монте-Карло**  = 2.66xxxx` (наближене значення)

**Абсолютна похибка** =< 0.006 (різниця)

*Метод Монте-Карло базується на генерації випадкових чисел - результат незначно змінюється при кожному запуску*

## Порівняльний аналіз

Метод Монте-Карло працює за принципом геометричної ймовірності. Прямокутник був заповнений випадковими точками у, що обмежує графік функції, далі підрахували частку точок, які потрапили під криву f(x).

**Спостереження:**
1.  **Точність:** Результат методу Монте-Карло дуже близький до результату функції quad.
2.  **Залежність від N:** Точність методу прямо пропорційна кількості точок - збільшення $N$ зменшує розкид значень, але збільшує час обчислень.
3.  **Продуктивність:** Для одновимірних інтегралів функція quad є швидшою та точнішою.

## Висновок
Алгоритм пошуку визначеного інтеграла методом Монте-Карло реалізовано коректно.
Порівняння з референсним значенням, отриманим функцією quad, підтверджує правильність розрахунків. Метод Монте-Карло довів свою працездатність як інструмент для чисельного інтегрування, хоча для простих функцій він поступається в точності детермінованим методам.
